## 🖥️ 1. Installing VM in Google IDX 

**Google IDX:**
[https://idx.google.com/](https://idx.google.com/) 

## ⚙ Steps 

1. Sign up on Google IDX.
2. Create a new workspace using this repository
3.Create a New Terminal 
4.Run the VM Setup CMD ( Below the text )
5.Select B
6.Then Select 4
7.Then Select 1 (Create a VM)
8.Then Select Ubuntu as Os
9.Then just enter VM name
10. Then just enter again on the Part of Hostname
11.Then Type Your Username ( Username : root )
12.Then Type Your Password ( Password : root )
13.Then type the Disk ( Disk : )
14.Then type the Memory ( Memory : )
15.Then click enter for the next steps
16.Wait,then enter after finish
17.Then select 2
18.Then select 1 
19.Wait Some time like 10 Min
10.Now your VM is Ready !!!
